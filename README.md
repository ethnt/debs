# Debs

[General Transit Feed Specification (GTFS)](https://gtfs.org/) parsing for Elixir.

Debs is named after [Eugene V. Debs](https://en.wikipedia.org/wiki/Eugene_V._Debs), one of the founders of the American Railway Union.

## Installation
